## Installation

Clone this repo:

https://github.com/Sydnee510/IPM-Pest-Portal

Direct:

    $ cd portal-backend

Install Dependicies:

    $ bundle install

Create Database:

    $ rails db:create 

Migrate Database:

    $ rails db:migrate

Start Server:

    $ rails s

 ## Framework

 Ruby on Rails
